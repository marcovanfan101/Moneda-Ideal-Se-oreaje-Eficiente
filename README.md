## 📊 Descripción del Proyecto
Este repositorio aloja el soporte computacional para la validación del modelo de **Señoreaje Eficiente**. El algoritmo sistematiza la extracción dinámica y el procesamiento de:

1. **Entropía de Masa Monetaria ($M_2$):** Medición del desorden informacional en la emisión.
2. **Eficiencia Operativa ($E_o$):** Análisis de fricción transaccional.
3. **Análisis Estadístico:** Correlaciones y validación de hipótesis (Tests Estadísticos).

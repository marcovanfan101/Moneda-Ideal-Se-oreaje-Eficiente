## 📊 Descripción del Proyecto
Este repositorio aloja el soporte computacional para la validación del modelo de **Señoreaje Eficiente**. El algoritmo sistematiza la extracción dinámica y el procesamiento de:

1. **Entropía de Masa Monetaria ($M_2$):** Medición del desorden informacional en la emisión primaria.
2. **Entropía de Valoración ($S_v$):** Análisis de la volatilidad logarítmica como pérdida de predictibilidad del valor de la divisa.
3. **Eficiencia Operativa ($E_o$):** Evaluación de la fricción transaccional y el spread cambiario.
4. **Análisis Estadístico:** Correlaciones cruzadas y validación de hipótesis (Tests Estadísticos de robustez).(https://colab.research.google.com/github/marcovanfan101/Moneda-Ideal-Se-oreaje-Eficiente/blob/main/Motor_Econometrico_Moneda_Ideal.ipynb)
